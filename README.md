# Geocoder de Locais (João Pessoa / Cabedelo)

1. Coloque seus locais em `locais.csv` com a coluna `Local`.[file:22]
2. Rode `npm install` e `npm run dev`.
3. Acesse `http://localhost:3000/api/geocode` para baixar `locais_geocodificados.csv`.
4. Faça deploy na Vercel apontando para este repositório.
5. Na Vercel, acesse `https://seu-projeto.vercel.app/api/geocode`.

Os dados de geocodificação vêm do Nominatim (OpenStreetMap); respeite a usage policy.[web:42][web:53][web:58]
